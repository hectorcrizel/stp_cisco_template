# Cisco STP Zabbix Template

![Zabbix](https://img.shields.io/badge/Zabbix-7.0%2B-red.svg)
![Cisco](https://img.shields.io/badge/Cisco-SNMP-blue.svg)

A specialized Zabbix template designed for deep monitoring of the Spanning Tree Protocol (STP) on Cisco hardware.

## 🚀 Key Features

*   **STP Discovery:** Automatically discovers STP instances and ports.
*   **Real-time Alerts:** Triggers for Designated Root Bridge changes and port state transitions.
*   **Friendly Names:** Uses `ifName` for clear interface identification in alerts and dashboards.
*   **LTS Compatible:** Optimized for Zabbix 7.0 LTS environments.

## 🛠️ Tech Stack

*   Zabbix (YAML Template)
*   SNMP
*   Cisco MIBs
